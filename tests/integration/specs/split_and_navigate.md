# Test: Split pane and navigate

## Steps

1. Press `n` to create a workspace named `dev`
2. Press prefix (ctrl+s), then `v` to split vertically
3. Read the screen and verify two panes are visible side by side, each with a shell prompt
4. Verify the sidebar shows a pane count indicator like `[2]`
5. Press prefix (ctrl+s), then `h` to navigate to the left pane
6. Press prefix (ctrl+s), then `l` to navigate back to the right pane
7. Verify the thick border (focused indicator) moved between panes
